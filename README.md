一、likab框架简介

likab框架提供高可用RMI远程调用，基于java rmi和zookeeper构建成的高可用远程通讯框架。

------
二、各层说明
> * likab-core — 核心包，提供likab各种核心功能
> * likab-loadbalance — 负载均衡层
> * likab-registry — 服务注册层
> * likab-remote —远程调用层
